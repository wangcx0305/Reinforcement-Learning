All the codes are implemented in Python 3.5

mdp.py: code to generate a MDP class
mdp_opt.py: code to optimize the MDP using policy iteration method
parking_MDP_creation.py: code to generate a parking MDP class
main.py: code to get final result using mdp.py, mdp_opt.py and parking_MDP_creation.py

value_MDP1_beta01.txt, policy_MDP1_beta01.txt: value function and optimal policy derived from MDP1 with beta = 0.1
value_MDP1_beta09.txt, policy_MDP1_beta09.txt: value function and optimal policy derived from MDP1 with beta = 0.9

value_MDP2_beta01.txt, policy_MDP2_beta01.txt: value function and optimal policy derived from MDP2 with beta = 0.1
value_MDP2_beta09.txt, policy_MDP2_beta09.txt: value function and optimal policy derived from MDP2 with beta = 0.9

P_MDP1.txt, P_MDP1_value.txt, P_MDP1_policy.txt: parking_MDP_1 input, value function and optimal policy 
P_MDP2.txt, P_MDP2_value.txt, P_MDP2_policy.txt: parking_MDP_2 input, value function and optimal policy
