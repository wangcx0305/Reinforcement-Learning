All the codes are implemented in Python 3.5

mdp.py: code to generate a MDP class
mdp_opt.py: code to optimize the MDP using value iteration method
main.py: code to get result using mdp.py and mdp_opt.py

MDP_example.txt, Value_example.txt, Policy_example.txt: MDP, value function and policy for the example in homework when H = 10

MDP_self.txt, Value_self.txt, Policy_self.txt: MDP, value function and policy for the self-defined MDP in homework when H = 10

MDP_1.txt, Value_1.txt, Policy_1.txt: MDP, value function and policy for MDP1 in homework when H = 10

MDP_2.txt, Value_2.txt, Policy_2.txt: MDP, value function and policy for the MDP2 in homework when H = 10
